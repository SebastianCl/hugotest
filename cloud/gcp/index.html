Google Cloud

Utilizamos los diferentes servicios de GCP para crear desarrollos a la medida.

Google Cloud Platform es una plataforma que ha reunido todas las aplicaciones de desarrollo web que Google estaba ofreciendo por separado. Es utilizada para crear ciertos tipos de soluciones a través de la tecnología almacenada en la nube y permite por ejemplo destacar la rapidez y la escalabilidad de su infraestructura en las aplicaciones del buscador. (Tomado de Wikipedia).

App Engine

Es ideal para desplegar aplicaciones web o también se puede desplegar el Backend de una aplicación. ¿De qué manera ayuda Google? Bueno, con App Engine no estoy trabajando con un sistema operativo, Google se encarga de abstraer la capa del SO y yo solo debo pensar en que estoy trabajando sobre un servidor en el que se van a ejecutar las aplicaciones que escriba con alguno de los lenguajes que soporta App Engine. También revuelve el escalamiento, balanceo de cargas, trabajar con múltiples veriones…

Viene en dos tipos de ambientes, Standard y Flexible. El ambiente Standard permite trabajar con cuatros runtimes que soportan los lenguajes como Go, Phyton, PHP y Java; mientras el ambiente Flexible ofrece estos mismos lenguajes además de NodeJS, .Net Core, Ruby y Docker.

BigQuery

Es un motor de base de datos columnar, significa que principalmente accede a la información que tengamos almacenada en tablas a manera de columnas. La diferencia mas importante es que se puede acceder a columnas especificas sin traer campos o registros completos, esto provee una gran facilidad en cuanto a ácueo de información y sobre todo en los tiempos de respuesta. El caso de uso principal de BigQuery es generar un data warehouse en la nube con algunas características importantes, siendo un servicio totalmente administrado por Google.

Cloud Datastore

Este es un directorio de almacenamiento de base de datos para información no estructura, comúnmente llamada NoSQL. Es una base de datos totalmente escalable que puede hacer el escalamiento automatico, significa que dependiendo del volumen transaccional de información o de consultas y escrituras que pueda tener, el servicio tiene la autonomía para crecer o decrecer dependiendo de la capacidad de computo requerida para responder a las solicitudes.

Cloud Storage

Es una de las opciones que nos provee GCP para el almacenamiento de datos. Permite generar un alto desempeño de almacenamiento y de consultas de información, su administración es muy sencilla ya que es un servicio administrado por Google, solo nos preocupamos por seleccionar el tipo de servicio o la disponibilidad de la información que almacenamos en Storage. No es como un sistema de archivos, es principalmente para usar la información por medio de hipervínculos. Una de las opciones interesantes que contempla Cloud Storage es que toda la información que viaja o es almacenada en Cloud Storage siempre va a ser encriptada.
